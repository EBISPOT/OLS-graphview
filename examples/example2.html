Demonstration of the ontology visualisation: example 2

Example 2 demonstrates some more options, the additional windows (ButtonBox, Infowindow, showLegend) are deactivated and some other changes are made through the options e.g. the automatic Cluster Threshold as well as the symbole for parent clusters and other things (mass for nodes! Colors!) are changed. However, all changes are made in the networkOptions object and no changes to the visoptions are made yet.
