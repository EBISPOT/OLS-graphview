Demonstration of the Ontology visualisation example 3

Example 3 shows the ability of changing vis options. Therefore the shape of the nodes, as well as their border and mass are changed. Also the edges have their arrows not in the middle, they are dashed and their width is set to 10 if an edge is selected.
!This example is started with an hierarchical layout. If changes to the layout options are made, it is suggested to DEACTIVATE the buttonBox as well. Changes to the layout options brake the default button to switch layouts. If you want to use different layout options, please also implement an own function to switch between layouts.!
